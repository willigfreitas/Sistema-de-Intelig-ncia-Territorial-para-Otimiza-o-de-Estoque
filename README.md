# Sistema-de-Intelig-ncia-Territorial-para-Otimiza-o-de-Estoque
Projeto de Inteligência Territorial aplicado à otimização de estoque em rede de concessionárias. Integração entre PostgreSQL/PostGIS e QGIS para análise de áreas de influência, processamento geoespacial e automação de regras de negócio via triggers e funções PL/pgSQL..
